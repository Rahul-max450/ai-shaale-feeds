ಅಧ್ಯಾಯ 3: ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಜೋಡಿಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಎರಡು ಚರಾಕ್ಷರಗಳನ್ನು (x ಮತ್ತು y) ಹೊಂದಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಜೋಡಿಗಳನ್ನು ಹೇಗೆ ಗುರುತಿಸುವುದು, ಅವುಗಳ ನಕ್ಷೆಯನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಮತ್ತು ಬೀಜಗಣಿತದ ವಿಧಾನಗಳಾದ 'ಆದೇಶ ವಿಧಾನ' ಮತ್ತು 'ವರ್ಜಿಸುವ ವಿಧಾನ'ಗಳನ್ನು ಬಳಸಿ ಅವುಗಳಿಗೆ ಪರಿಹಾರಗಳನ್ನು ಹೇಗೆ ಕಂಡುಹಿಡಿಯುವುದು ಎಂಬುದನ್ನು ವಿವರಿಸುತ್ತದೆ. ರೇಖೆಗಳ ಸ್ವಭಾವ (ಛೇದಿಸುವ, ಸಮಾಂತರ ಅಥವಾ ಐಕ್ಯಗೊಳ್ಳುವ) ಮತ್ತು ಅವುಗಳ ಪರಿಹಾರಗಳ ಅಸ್ತಿತ್ವದ ಬಗ್ಗೆ ಈ ಅಧ್ಯಾಯವು ಬೆಳಕು ಚೆಲ್ಲುತ್ತದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

ಸಾಮಾನ್ಯ ರೂಪ: a₁x + b₁y + c₁ = 0 ಮತ್ತು a₂x + b₂y + c₂ = 0

  • ಛೇದಿಸುವ ರೇಖೆಗಳು (ಅನನ್ಯ ಪರಿಹಾರ): a₁/a₂ ≠ b₁/b₂
  • ಸಮಾಂತರ ರೇಖೆಗಳು (ಪರಿಹಾರವಿಲ್ಲ): a₁/a₂ = b₁/b₂ ≠ c₁/c₂
  • ಐಕ್ಯಗೊಳ್ಳುವ ರೇಖೆಗಳು (ಅಪರಿಮಿತ ಪರಿಹಾರಗಳು): a₁/a₂ = b₁/b₂ = c₁/c₂

ಅಭ್ಯಾಸ 3.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1(i): 10 ವಿದ್ಯಾರ್ಥಿಗಳು ರಸಪ್ರಶ್ನೆಯಲ್ಲಿ ಭಾಗವಹಿಸಿದ್ದಾರೆ. ಹುಡುಗಿಯರ ಸಂಖ್ಯೆ ಹುಡುಗರ ಸಂಖ್ಯೆಗಿಂತ 4 ಹೆಚ್ಚಿದ್ದರೆ, ಹುಡುಗ ಮತ್ತು ಹುಡುಗಿಯರ ಸಂಖ್ಯೆ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: ಹುಡುಗರ ಸಂಖ್ಯೆ = x, ಹುಡುಗಿಯರ ಸಂಖ್ಯೆ = y.
x + y = 10 (1)
y = x + 4 (2)
ಆದೇಶಿಸಿದಾಗ: x + (x + 4) = 10 => 2x = 6 => x = 3.
y = 3 + 4 = 7. ಹುಡುಗರು 3, ಹುಡುಗಿಯರು 7.

ಪ್ರಶ್ನೆ 1(ii): 5 ಪೆನ್ಸಿಲ್ ಮತ್ತು 7 ಪೆನ್ನುಗಳ ಬೆಲೆ ₹50. 7 ಪೆನ್ಸಿಲ್ ಮತ್ತು 5 ಪೆನ್ನುಗಳ ಬೆಲೆ ₹46. ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: ಪೆನ್ಸಿಲ್ ಬೆಲೆ = x, ಪೆನ್ ಬೆಲೆ = y.
5x + 7y = 50 (1)
7x + 5y = 46 (2)
ವರ್ಜಿಸುವ ವಿಧಾನ: (1)x7 ಮತ್ತು (2)x5 ಮಾಡಿದಾಗ:
35x + 49y = 350
35x + 25y = 230
ಕಳೆದಾಗ: 24y = 120 => y = 5.
x = (50 - 35)/5 = 3. ಪೆನ್ಸಿಲ್ ₹3, ಪೆನ್ ₹5.

ಅಭ್ಯಾಸ 3.2 - ಪ್ರಮುಖ ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 2: 2x + 3y = 11 ಮತ್ತು 2x - 4y = -24 ಅನ್ನು ಬಿಡಿಸಿ, y = mx + 3 ರಲ್ಲಿ m ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
2x + 3y = 11 (1)
2x - 4y = -24 (2)
(1) - (2) ಮಾಡಿದಾಗ: 7y = 35 => y = 5.
2x + 3(5) = 11 => 2x = -4 => x = -2.
y = mx + 3 => 5 = m(-2) + 3 => 2 = -2m => m = -1.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ನಕ್ಷೆ ಬಿಡಿಸುವಾಗ ಅಕ್ಷಗಳ ಮೇಲೆ ಸರಿಯಾದ ಅಳತೆ (Scale) ಗುರುತಿಸಿ.
  • ವರ್ಜಿಸುವ ವಿಧಾನದಲ್ಲಿ (Elimination Method) ಗುಣಕಗಳನ್ನು ಸಮಗೊಳಿಸುವಾಗ ಚಿಹ್ನೆಗಳನ್ನು (Signs) ಎಚ್ಚರಿಕೆಯಿಂದ ಗಮನಿಸಿ.
  • ಪರಿಹಾರ ಸಿಕ್ಕಿದ ನಂತರ, ಅದನ್ನು ಮೂಲ ಸಮೀಕರಣದಲ್ಲಿ ಆದೇಶಿಸಿ ತಾಳೆ ನೋಡಿ.
  • a₁/a₂ ಅನುಪಾತಗಳನ್ನು ಹೋಲಿಸುವಾಗ c₁/c₂ ಅನ್ನು ಮರೆಯಬೇಡಿ, ವಿಶೇಷವಾಗಿ ಸಮಾಂತರ ಮತ್ತು ಐಕ್ಯಗೊಳ್ಳುವ ರೇಖೆಗಳ ಸಂದರ್ಭದಲ್ಲಿ.
