ಅಧ್ಯಾಯ 9: ತ್ರಿಕೋನಮಿತಿಯ ಕೆಲವು ಅನ್ವಯಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ತ್ರಿಕೋನಮಿತಿಯ ಮೂಲಭೂತ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಬಳಸಿ, ಎತ್ತರ ಮತ್ತು ದೂರಗಳನ್ನು ನೇರವಾಗಿ ಅಳೆಯದೆ ಹೇಗೆ ಕಂಡುಹಿಡಿಯಬಹುದು ಎಂಬುದನ್ನು ವಿವರಿಸುತ್ತದೆ. ಇದರಲ್ಲಿ ದೃಷ್ಟಿ ರೇಖೆ, ಕ್ಷಿತಿಜ ರೇಖೆ, ಉನ್ನತ ಕೋನ (Angle of Elevation) ಮತ್ತು ಅವನತ ಕೋನ (Angle of Depression) ಎಂಬ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಲಂಬಕೋನ ತ್ರಿಭುಜಗಳ ಮೂಲಕ ಸಮಸ್ಯೆಗಳನ್ನು ಬಿಡಿಸಲಾಗುತ್ತದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • tan θ = ಅಭಿಮುಖ ಬಾಹು / ಪಾರ್ಶ್ವ ಬಾಹು
  • cot θ = ಪಾರ್ಶ್ವ ಬಾಹು / ಅಭಿಮುಖ ಬಾಹು
  • sin θ = ಅಭಿಮುಖ ಬಾಹು / ವಿಕರ್ಣ
  • cos θ = ಪಾರ್ಶ್ವ ಬಾಹು / ವಿಕರ್ಣ
  • ಉನ್ನತ ಕೋನ: ವೀಕ್ಷಕನು ವಸ್ತುವನ್ನು ನೋಡಲು ತಲೆಯನ್ನು ಮೇಲೆತ್ತಿದಾಗ ಉಂಟಾಗುವ ಕೋನ.
  • ಅವನತ ಕೋನ: ವೀಕ್ಷಕನು ವಸ್ತುವನ್ನು ನೋಡಲು ತಲೆಯನ್ನು ಕೆಳಗೆ ಬಾಗಿಸಿದಾಗ ಉಂಟಾಗುವ ಕೋನ.
  • ಪ್ರಮುಖ ಬೆಲೆಗಳು: tan 30° = 1/√3, tan 45° = 1, tan 60° = √3.

ಅಭ್ಯಾಸ 9.1 ರ ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಒಬ್ಬ ಸರ್ಕಸ್ ಕಲಾವಿದನು, ನೇರ ಕಂಬದಿಂದ ಬಿಗಿಯಾಗಿ ನೆಲಕ್ಕೆ ಕಟ್ಟಿರುವ 20 m ಉದ್ದದ ಹಗ್ಗದ ಮೇಲೆ ಹತ್ತುತ್ತಿದ್ದಾನೆ. ನೆಲದೊಂದಿಗೆ ಹಗ್ಗವು ಉಂಟುಮಾಡಿದ ಕೋನ 30° ಆದರೆ, ಕಂಬದ ಎತ್ತರ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: ತ್ರಿಭುಜದಲ್ಲಿ, sin 30° = ಎತ್ತರ (h) / ಹಗ್ಗದ ಉದ್ದ (20).
1/2 = h / 20 => h = 20/2 = 10 m. ಕಂಬದ ಎತ್ತರ 10 m.

ಪ್ರಶ್ನೆ 2: ಬಿರುಗಾಳಿಗೆ ಮರ ಮುರಿದು ನೆಲಕ್ಕೆ ತಾಗಿದಾಗ 30° ಕೋನ ಉಂಟುಮಾಡಿದೆ. ಮರದ ತುದಿ ಬುಡದಿಂದ 8 m ದೂರದಲ್ಲಿದೆ. ಮರದ ಒಟ್ಟು ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 30° = h1 / 8 => h1 = 8/√3. cos 30° = 8 / ವಿಕರ್ಣ (h2) => h2 = 16/√3.
ಒಟ್ಟು ಎತ್ತರ = (8+16)/√3 = 24/√3 = 8√3 m.

ಪ್ರಶ್ನೆ 3: 5 ವರ್ಷದೊಳಗಿನ ಮಕ್ಕಳಿಗೆ 1.5 m ಎತ್ತರ ಮತ್ತು 30° ಕೋನ, ಹಿರಿಯರಿಗೆ 3 m ಎತ್ತರ ಮತ್ತು 60° ಕೋನದ ಜಾರುಬಂಡೆ. ಜಾರುಬಂಡೆಯ ಉದ್ದ ಎಷ್ಟು?

ಪರಿಹಾರ: ಸಣ್ಣ ಮಕ್ಕಳಿಗೆ: sin 30° = 1.5 / L1 => L1 = 3 m.
ದೊಡ್ಡ ಮಕ್ಕಳಿಗೆ: sin 60° = 3 / L2 => L2 = 3 / (√3/2) = 2√3 m.

ಪ್ರಶ್ನೆ 4: ಗೋಪುರದ ಪಾದದಿಂದ 30 m ದೂರದ ಬಿಂದುವಿನಿಂದ ಉನ್ನತ ಕೋನ 30°. ಗೋಪುರದ ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 30° = h / 30 => 1/√3 = h / 30 => h = 30/√3 = 10√3 m.

ಪ್ರಶ್ನೆ 5: 60 m ಎತ್ತರದಲ್ಲಿ ಹಾರುವ ಗಾಳಿಪಟದ ದಾರದ ಉದ್ದ ಕಂಡುಹಿಡಿಯಿರಿ (ಕೋನ 60°).

ಪರಿಹಾರ: sin 60° = 60 / L => √3/2 = 60 / L => L = 120/√3 = 40√3 m.

ಪ್ರಶ್ನೆ 6: 1.5 m ಎತ್ತರದ ಹುಡುಗ 30 m ಎತ್ತರದ ಕಟ್ಟಡದ ಕಡೆಗೆ ನಡೆದಾಗ ಕೋನ 30° ದಿಂದ 60° ಆಗುತ್ತದೆ. ಅವನು ನಡೆದ ದೂರ ಎಷ್ಟು?

ಪರಿಹಾರ: ಎತ್ತರ = 30 - 1.5 = 28.5 m. tan 30° = 28.5 / x1, tan 60° = 28.5 / x2. ನಡೆದ ದೂರ = x1 - x2 = 28.5(√3 - 1/√3) = 19√3 m.

ಪ್ರಶ್ನೆ 7: 20 m ಎತ್ತರದ ಕಟ್ಟಡದ ಮೇಲಿನ ಗೋಪುರದ ಉನ್ನತ ಕೋನಗಳು 60° ಮತ್ತು 45°. ಗೋಪುರದ ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 45° = 20 / ದೂರ => ದೂರ = 20 m. tan 60° = (20+h) / 20 => 20√3 = 20+h => h = 20(√3-1) m.

ಪ್ರಶ್ನೆ 8: 1.6 m ಪ್ರತಿಮೆಯ ಉನ್ನತ ಕೋನ 60° ಮತ್ತು ಪೀಠದ ಕೋನ 45°. ಪೀಠದ ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 45° = h / d => d = h. tan 60° = (h+1.6) / h => h√3 = h+1.6 => h = 1.6 / (√3-1) m.

ಪ್ರಶ್ನೆ 9: ಗೋಪುರದ ಎತ್ತರ 50 m, ಕಟ್ಟಡದ ಉನ್ನತ ಕೋನ 30°, ಗೋಪುರದ ಉನ್ನತ ಕೋನ 60°. ಕಟ್ಟಡದ ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 60° = 50 / d => d = 50/√3. tan 30° = h / d => h = (50/√3) * (1/√3) = 50/3 = 16.67 m.

ಪ್ರಶ್ನೆ 10: 80 ಅಡಿ ರಸ್ತೆಯ ಇಕ್ಕೆಲಗಳಲ್ಲಿ ಸಮಾನ ಎತ್ತರದ ಕಂಬಗಳು. ಕೋನಗಳು 60° ಮತ್ತು 30°. ಕಂಬದ ಎತ್ತರ ಮತ್ತು ದೂರ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: tan 60° = h/x, tan 30° = h/(80-x). ಬಿಡಿಸಿದಾಗ h = 20√3 ಅಡಿ.

ಪ್ರಶ್ನೆ 11: ಗೋಪುರದ ಎತ್ತರ ಮತ್ತು ಕಾಲುವೆಯ ಅಗಲ ಕಂಡುಹಿಡಿಯಿರಿ (ಕೋನಗಳು 60° ಮತ್ತು 30°, ಅಂತರ 20 m).

ಪರಿಹಾರ: tan 60° = h/x, tan 30° = h/(x+20). h = 10√3 m, ಅಗಲ = 10 m.

ಪ್ರಶ್ನೆ 12: 7 m ಕಟ್ಟಡದ ಮೇಲಿನಿಂದ ಗೋಪುರದ ಉನ್ನತ ಕೋನ 60° ಮತ್ತು ಅವನತ ಕೋನ 45°. ಗೋಪುರದ ಎತ್ತರ ಎಷ್ಟು?

ಪರಿಹಾರ: tan 45° = 7/d => d = 7. tan 60° = h'/7 => h' = 7√3. ಒಟ್ಟು ಎತ್ತರ = 7(√3+1) m.

ಪ್ರಶ್ನೆ 13: 75 m ದೀಪಸ್ತಂಭದಿಂದ ಹಡಗುಗಳ ಅಂತರ (ಕೋನಗಳು 30° ಮತ್ತು 45°).

ಪರಿಹಾರ: tan 45° = 75/d1 => d1 = 75. tan 30° = 75/d2 => d2 = 75√3. ಅಂತರ = 75(√3-1) m.

ಪ್ರಶ್ನೆ 14: 1.2 m ಹುಡುಗಿ 88.2 m ಎತ್ತರದ ಬಲೂನ್ ನೋಡುತ್ತಿದ್ದಾಳೆ (ಕೋನ 60° ದಿಂದ 30°). ಬಲೂನ್ ಚಲಿಸಿದ ದೂರ ಎಷ್ಟು?

ಪರಿಹಾರ: ಎತ್ತರ = 88.2 - 1.2 = 87 m. tan 60° = 87/x1, tan 30° = 87/x2. ದೂರ = 87(√3 - 1/√3) = 58√3 m.

ಪ್ರಶ್ನೆ 15: ಗೋಪುರದ ಪಾದಕ್ಕೆ ಕಾರು ಬರುವ ಸಮಯ (ಕೋನ 30° ದಿಂದ 60°, ಸಮಯ 6 ಸೆಕೆಂಡ್).

ಪರಿಹಾರ: tan 30° = h/d1, tan 60° = h/d2. d1 = 3*d2. ಕಾರು 2 ಭಾಗದ ದೂರಕ್ಕೆ 6 ಸೆಕೆಂಡ್ ತೆಗೆದುಕೊಂಡರೆ, ಉಳಿದ ದೂರಕ್ಕೆ 3 ಸೆಕೆಂಡ್ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಚಿತ್ರವನ್ನು ಸರಿಯಾಗಿ ಬಿಡಿಸುವುದು ಅರ್ಧ ಪರಿಹಾರವಿದ್ದಂತೆ.
  • ಲಂಬಕೋನ ತ್ರಿಭುಜದ ಬಾಹುಗಳನ್ನು ಗುರುತಿಸುವಾಗ ಎಚ್ಚರವಿರಲಿ.
  • √3 = 1.732 ಎಂದು ಬಳಸುವಾಗ ಲೆಕ್ಕಾಚಾರದಲ್ಲಿ ನಿಖರತೆ ಇರಲಿ.
  • ಅವನತ ಕೋನವು ಪರ್ಯಾಯ ಕೋನಗಳ ನಿಯಮದಂತೆ ಉನ್ನತ ಕೋನಕ್ಕೆ ಸಮನಾಗಿರುತ್ತದೆ ಎಂಬುದನ್ನು ಮರೆಯದಿರಿ.
