ಅಧ್ಯಾಯ 13: ಸಾಂಖ್ಯಶಾಸ್ತ್ರ (Statistics)

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ವರ್ಗೀಕೃತ ದತ್ತಾಂಶಗಳ ಕೇಂದ್ರೀಯ ಪ್ರವೃತ್ತಿಯ ಅಳತೆಗಳಾದ ಸರಾಸರಿ (Mean), ಮಧ್ಯಾಂಕ (Median) ಮತ್ತು ಬಹುಳಕ (Mode) ಗಳ ಬಗ್ಗೆ ವಿವರಿಸುತ್ತದೆ. ದತ್ತಾಂಶಗಳನ್ನು ವರ್ಗೀಕೃತ ರೂಪಕ್ಕೆ ಬದಲಾಯಿಸುವುದು, ಅವುಗಳಿಗೆ ಸೂಕ್ತವಾದ ಸೂತ್ರಗಳನ್ನು ಅನ್ವಯಿಸಿ ಮೌಲ್ಯಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು ಮತ್ತು ನೈಜ ಜೀವನದ ಸಮಸ್ಯೆಗಳಿಗೆ ಇವುಗಳನ್ನು ಹೇಗೆ ಅನ್ವಯಿಸಬೇಕು ಎಂಬುದನ್ನು ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಕಲಿಯುತ್ತೇವೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಸರಾಸರಿ (Mean):
    • ನೇರ ವಿಧಾನ: x̄ = Σ(fᵢxᵢ) / Σfᵢ
    • ಊಹಿಸಿದ ಸರಾಸರಿ ವಿಧಾನ: x̄ = a + Σ(fᵢdᵢ) / Σfᵢ (ಇಲ್ಲಿ dᵢ = xᵢ - a)
    • ಹಂತ ವಿಚಲನ ವಿಧಾನ: x̄ = a + [Σ(fᵢuᵢ) / Σfᵢ] × h (ಇಲ್ಲಿ uᵢ = (xᵢ - a) / h)
  • ಬಹುಳಕ (Mode):
    • ಬಹುಳಕ = l + [(f₁ - f₀) / (2f₁ - f₀ - f₂)] × h
    • (l = ಬಹುಳಕ ವರ್ಗದ ಕೆಳಮಿತಿ, h = ವರ್ಗದ ಗಾತ್ರ, f₁ = ಬಹುಳಕ ವರ್ಗದ ಆವೃತ್ತಿ, f₀ = ಹಿಂದಿನ ವರ್ಗದ ಆವೃತ್ತಿ, f₂ = ಮುಂದಿನ ವರ್ಗದ ಆವೃತ್ತಿ)
  • ಮಧ್ಯಾಂಕ (Median):
    • ಮಧ್ಯಾಂಕ = l + [(n/2 - cf) / f] × h
    • (l = ಮಧ್ಯಾಂಕ ವರ್ಗದ ಕೆಳಮಿತಿ, n = ಒಟ್ಟು ಆವೃತ್ತಿ, cf = ಮಧ್ಯಾಂಕ ವರ್ಗದ ಹಿಂದಿನ ವರ್ಗದ ಸಂಚಿತ ಆವೃತ್ತಿ, f = ಮಧ್ಯಾಂಕ ವರ್ಗದ ಆವೃತ್ತಿ, h = ವರ್ಗದ ಗಾತ್ರ)
  • ಅನುಭವಾತ್ಮಕ ಸಂಬಂಧ: 3 ಮಧ್ಯಾಂಕ = ಬಹುಳಕ + 2 ಸರಾಸರಿ

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ವರ್ಗಾಂತರಗಳು ನಿರಂತರವಾಗಿಲ್ಲದಿದ್ದರೆ (ಉದಾ: 118-126, 127-135), ಮೊದಲು ಅವುಗಳನ್ನು ನಿರಂತರ ವರ್ಗಾಂತರಗಳಾಗಿ (117.5-126.5, 126.5-135.5) ಬದಲಾಯಿಸಿಕೊಳ್ಳಿ.
  • ಮಧ್ಯಾಂಕ ಕಂಡುಹಿಡಿಯುವಾಗ 'n/2' ಗಿಂತ ದೊಡ್ಡದಾದ ಮೊದಲ ಸಂಚಿತ ಆವೃತ್ತಿಯನ್ನು ಗುರುತಿಸುವುದು ಅತ್ಯಗತ್ಯ.
  • ಹಂತ ವಿಚಲನ ವಿಧಾನವನ್ನು ಬಳಸುವಾಗ 'h' (ವರ್ಗ ಗಾತ್ರ) ಸರಿಯಾಗಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
