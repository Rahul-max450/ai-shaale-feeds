ಅಧ್ಯಾಯ 2: ಬಹುಪದೋಕ್ತಿಗಳು (Polynomials)

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಬಹುಪದೋಕ್ತಿಗಳ ಪರಿಚಯ, ಅವುಗಳ ವಿಧಗಳು (ರೇಖಾತ್ಮಕ, ವರ್ಗ ಮತ್ತು ಘನ ಬಹುಪದೋಕ್ತಿಗಳು), ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳ ರೇಖಾಗಣಿತದ ಅರ್ಥ ಮತ್ತು ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳು ಹಾಗೂ ಸಹಗುಣಕಗಳ ನಡುವಿನ ಸಂಬಂಧವನ್ನು ವಿವರಿಸುತ್ತದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿ (ax + b): ಶೂನ್ಯತೆ = -b/a
  • ವರ್ಗ ಬಹುಪದೋಕ್ತಿ (ax² + bx + c):
    • ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ (α + β) = -b/a
    • ಶೂನ್ಯತೆಗಳ ಗುಣಲಬ್ಧ (αβ) = c/a
  • ಘನ ಬಹುಪದೋಕ್ತಿ (ax³ + bx² + cx + d):
    • ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ (α + β + γ) = -b/a
    • ಎರಡೆರಡು ಶೂನ್ಯತೆಗಳ ಗುಣಲಬ್ಧಗಳ ಮೊತ್ತ (αβ + βγ + γα) = c/a
    • ಶೂನ್ಯತೆಗಳ ಗುಣಲಬ್ಧ (αβγ) = -d/a

ಅಭ್ಯಾಸ 2.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: y = p(x) ನ ನಕ್ಷೆಗಳನ್ನು ಕೆಳಗೆ ನೀಡಿದ್ದು, p(x) ನ ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:

  • (i) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಎಲ್ಲಿಯೂ ಛೇದಿಸುವುದಿಲ್ಲ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 0.
  • (ii) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು 1 ಬಿಂದುವಿನಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 1.
  • (iii) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು 3 ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 3.
  • (iv) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು 2 ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 2.
  • (v) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು 4 ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 4.
  • (vi) ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು 3 ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಶೂನ್ಯತೆಗಳ ಸಂಖ್ಯೆ = 3.

ಅಭ್ಯಾಸ 2.2 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ವರ್ಗ ಬಹುಪದೋಕ್ತಿಗಳ ಶೂನ್ಯತೆಗಳನ್ನು ಕಂಡುಹಿಡಿದು, ಶೂನ್ಯತೆಗಳು ಮತ್ತು ಸಹಗುಣಕಗಳ ನಡುವಿನ ಸಂಬಂಧವನ್ನು ತಾಳೆ ನೋಡಿ.

(i) x² - 2x - 8
ಪರಿಹಾರ: x² - 4x + 2x - 8 = x(x-4) + 2(x-4) = (x+2)(x-4). ಶೂನ್ಯತೆಗಳು: -2, 4.
ಮೊತ್ತ: -2+4 = 2; -b/a = -(-2)/1 = 2. (ಸರಿ)
ಗುಣಲಬ್ಧ: -2*4 = -8; c/a = -8/1 = -8. (ಸರಿ)

(ii) 4s² - 4s + 1
ಪರಿಹಾರ: 4s² - 2s - 2s + 1 = 2s(2s-1) - 1(2s-1) = (2s-1)². ಶೂನ್ಯತೆಗಳು: 1/2, 1/2.
ಮೊತ್ತ: 1/2 + 1/2 = 1; -b/a = -(-4)/4 = 1. (ಸರಿ)
ಗುಣಲಬ್ಧ: 1/2 * 1/2 = 1/4; c/a = 1/4. (ಸರಿ)

(iii) 6x² - 7x - 3
ಪರಿಹಾರ: 6x² - 9x + 2x - 3 = 3x(2x-3) + 1(2x-3) = (3x+1)(2x-3). ಶೂನ್ಯತೆಗಳು: -1/3, 3/2.
ಮೊತ್ತ: -1/3 + 3/2 = 7/6; -b/a = -(-7)/6 = 7/6. (ಸರಿ)
ಗುಣಲಬ್ಧ: -1/3 * 3/2 = -1/2; c/a = -3/6 = -1/2. (ಸರಿ)

(iv) 4u² + 8u
ಪರಿಹಾರ: 4u(u+2). ಶೂನ್ಯತೆಗಳು: 0, -2.
ಮೊತ್ತ: 0 + (-2) = -2; -b/a = -8/4 = -2. (ಸರಿ)
ಗುಣಲಬ್ಧ: 0 * -2 = 0; c/a = 0/4 = 0. (ಸರಿ)

(v) t² - 15
ಪರಿಹಾರ: (t - √15)(t + √15). ಶೂನ್ಯತೆಗಳು: √15, -√15.
ಮೊತ್ತ: √15 - √15 = 0; -b/a = 0/1 = 0. (ಸರಿ)
ಗುಣಲಬ್ಧ: √15 * -√15 = -15; c/a = -15/1 = -15. (ಸರಿ)

(vi) 3x² - x - 4
ಪರಿಹಾರ: 3x² - 4x + 3x - 4 = x(3x-4) + 1(3x-4) = (x+1)(3x-4). ಶೂನ್ಯತೆಗಳು: -1, 4/3.
ಮೊತ್ತ: -1 + 4/3 = 1/3; -b/a = -(-1)/3 = 1/3. (ಸರಿ)
ಗುಣಲಬ್ಧ: -1 * 4/3 = -4/3; c/a = -4/3. (ಸರಿ)

ಪ್ರಶ್ನೆ 2: ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ ಮತ್ತು ಗುಣಲಬ್ಧ ನೀಡಲಾಗಿದೆ, ವರ್ಗ ಬಹುಪದೋಕ್ತಿ ರಚಿಸಿ (ಸೂತ್ರ: x² - (ಮೊತ್ತ)x + ಗುಣಲಬ್ಧ).

  • (i) 1/4, -1: x² - 1/4x - 1 = 4x² - x - 4
  • (ii) √2, 1/3: x² - √2x + 1/3 = 3x² - 3√2x + 1
  • (iii) 0, √5: x² - 0x + √5 = x² + √5
  • (iv) 1, 1: x² - x + 1
  • (v) -1/4, 1/4: x² + 1/4x + 1/4 = 4x² + x + 1
  • (vi) 4, 1: x² - 4x + 1

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಬಹುಪದೋಕ್ತಿಯನ್ನು ax² + bx + c ರೂಪದಲ್ಲಿ ಬರೆಯುವಾಗ ಚಿಹ್ನೆಗಳನ್ನು (signs) ಎಚ್ಚರಿಕೆಯಿಂದ ಗಮನಿಸಿ.
  • ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ ಮತ್ತು ಗುಣಲಬ್ಧದ ಸೂತ್ರಗಳಲ್ಲಿನ 'a' ಬೆಲೆಯು ಎರಡೂ ಕಡೆ ಒಂದೇ ಆಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
  • ನಕ್ಷೆಗಳನ್ನು ನೋಡುವಾಗ, ರೇಖೆಯು x-ಅಕ್ಷವನ್ನು ಎಷ್ಟು ಬಾರಿ ಸ್ಪರ್ಶಿಸುತ್ತದೆ ಅಥವಾ ಛೇದಿಸುತ್ತದೆ ಎಂಬುದನ್ನು ಮಾತ್ರ ಎಣಿಸಿ.
