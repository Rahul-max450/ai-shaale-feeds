ಅಧ್ಯಾಯ 4: ವರ್ಗ ಸಮೀಕರಣಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ವರ್ಗ ಸಮೀಕರಣಗಳ ಪರಿಚಯ, ಅವುಗಳ ಆದರ್ಶ ರೂಪ (ax² + bx + c = 0), ಮತ್ತು ಅವುಗಳನ್ನು ಬಿಡಿಸುವ ವಿವಿಧ ವಿಧಾನಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ವರ್ಗ ಸಮೀಕರಣಗಳನ್ನು ಅಪವರ್ತನ ವಿಧಾನ ಮತ್ತು ಸೂತ್ರದ ಸಹಾಯದಿಂದ ಹೇಗೆ ಪರಿಹರಿಸುವುದು ಎಂಬುದನ್ನು ಇಲ್ಲಿ ಕಲಿಯಬಹುದು. ಅಲ್ಲದೆ, ವರ್ಗ ಸಮೀಕರಣಗಳ ಮೂಲಗಳ ಸ್ವಭಾವವನ್ನು ನಿರ್ಧರಿಸುವ 'ಶೋಧಕ' (Discriminant) ಪರಿಕಲ್ಪನೆಯನ್ನು ಮತ್ತು ದೈನಂದಿನ ಜೀವನದ ಸಮಸ್ಯೆಗಳಿಗೆ ಇವುಗಳ ಅನ್ವಯವನ್ನು ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಚರ್ಚಿಸಲಾಗಿದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ವರ್ಗ ಸಮೀಕರಣದ ಆದರ್ಶ ರೂಪ: ax² + bx + c = 0, ಇಲ್ಲಿ a ≠ 0.
  • ವರ್ಗ ಸಮೀಕರಣದ ಸೂತ್ರ: x = [-b ± √(b² - 4ac)] / 2a
  • ಶೋಧಕ (Discriminant): D = b² - 4ac
  • ಮೂಲಗಳ ಸ್ವಭಾವ:
    • b² - 4ac > 0: ಎರಡು ಭಿನ್ನವಾದ ವಾಸ್ತವ ಮೂಲಗಳು.
    • b² - 4ac = 0: ಎರಡು ಸಮನಾದ ವಾಸ್ತವ ಮೂಲಗಳು.
    • b² - 4ac < 0: ವಾಸ್ತವ ಮೂಲಗಳಿಲ್ಲ.

ಅಭ್ಯಾಸ 4.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಕೆಳಗಿನವುಗಳು ವರ್ಗ ಸಮೀಕರಣಗಳೇ ಎಂದು ಪರೀಕ್ಷಿಸಿ.

  1. (x+1)² = 2(x-3) ⇒ x² + 2x + 1 = 2x - 6 ⇒ x² + 7 = 0. ಹೌದು, ಇದು ವರ್ಗ ಸಮೀಕರಣ.
  2. x² - 2x = (-2)(3-x) ⇒ x² - 2x = -6 + 2x ⇒ x² - 4x + 6 = 0. ಹೌದು, ಇದು ವರ್ಗ ಸಮೀಕರಣ.
  3. (x-2)(x+1) = (x-1)(x+3) ⇒ x² - x - 2 = x² + 2x - 3 ⇒ 3x - 1 = 0. ಇದು ವರ್ಗ ಸಮೀಕರಣವಲ್ಲ.
  4. (x-3)(2x+1) = x(x+5) ⇒ 2x² - 5x - 3 = x² + 5x ⇒ x² - 10x - 3 = 0. ಹೌದು, ಇದು ವರ್ಗ ಸಮೀಕರಣ.
  5. (2x-1)(x-3) = (x+5)(x-1) ⇒ 2x² - 7x + 3 = x² + 4x - 5 ⇒ x² - 11x + 8 = 0. ಹೌದು, ಇದು ವರ್ಗ ಸಮೀಕರಣ.
  6. x² + 3x + 1 = (x-2)² ⇒ x² + 3x + 1 = x² - 4x + 4 ⇒ 7x - 3 = 0. ಇದು ವರ್ಗ ಸಮೀಕರಣವಲ್ಲ.
  7. (x+2)³ = 2x(x²-1) ⇒ x³ + 6x² + 12x + 8 = 2x³ - 2x ⇒ x³ - 6x² - 14x - 8 = 0. ಇದು ವರ್ಗ ಸಮೀಕರಣವಲ್ಲ (ಘಾತ 3 ಇದೆ).
  8. x³ - 4x² - x + 1 = (x-2)³ ⇒ x³ - 4x² - x + 1 = x³ - 6x² + 12x - 8 ⇒ 2x² - 13x + 9 = 0. ಹೌದು, ಇದು ವರ್ಗ ಸಮೀಕರಣ.

ಅಭ್ಯಾಸ 4.2 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಅಪವರ್ತನ ವಿಧಾನದಿಂದ ಮೂಲಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.

  1. x² - 3x - 10 = 0 ⇒ x² - 5x + 2x - 10 = 0 ⇒ (x-5)(x+2) = 0. ಮೂಲಗಳು: 5, -2.
  2. 2x² + x - 6 = 0 ⇒ 2x² + 4x - 3x - 6 = 0 ⇒ 2x(x+2) - 3(x+2) = 0. ಮೂಲಗಳು: 3/2, -2.
  3. √2x² + 7x + 5√2 = 0 ⇒ √2x² + 2x + 5x + 5√2 = 0 ⇒ √2x(x+√2) + 5(x+√2) = 0. ಮೂಲಗಳು: -√2, -5/√2.
  4. 2x² - x + 1/8 = 0 ⇒ 16x² - 8x + 1 = 0 ⇒ (4x-1)² = 0. ಮೂಲಗಳು: 1/4, 1/4.
  5. 100x² - 20x + 1 = 0 ⇒ (10x-1)² = 0. ಮೂಲಗಳು: 1/10, 1/10.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಸಮೀಕರಣವನ್ನು ಪರೀಕ್ಷಿಸುವಾಗ ಯಾವಾಗಲೂ ಅದನ್ನು ಸರಳೀಕರಿಸಿ ax² + bx + c = 0 ರೂಪಕ್ಕೆ ತನ್ನಿ.
  • ಮಧ್ಯದ ಪದವನ್ನು ವಿಭಜಿಸುವಾಗ (Splitting the middle term), ಗುಣಲಬ್ಧವು 'a × c' ಗೆ ಸಮನಾಗಿರಬೇಕು ಎಂಬುದನ್ನು ಮರೆಯದಿರಿ.
  • ಶೋಧಕ (D) ಋಣಾತ್ಮಕವಾಗಿದ್ದರೆ, ವಾಸ್ತವ ಮೂಲಗಳು ಇರುವುದಿಲ್ಲ ಎಂದು ತಕ್ಷಣ ಗುರುತಿಸಿ.
  • ಲೆಕ್ಕ ಮಾಡುವಾಗ ಚಿಹ್ನೆಗಳ (+/-) ಬಗ್ಗೆ ಅತ್ಯಂತ ಎಚ್ಚರವಿರಲಿ, ಇದುವೇ ಹೆಚ್ಚಿನ ವಿದ್ಯಾರ್ಥಿಗಳು ಮಾಡುವ ಸಾಮಾನ್ಯ ತಪ್ಪು.
