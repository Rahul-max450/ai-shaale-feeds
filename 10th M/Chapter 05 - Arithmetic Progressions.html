ಅಧ್ಯಾಯ 5: ಸಮಾಂತರ ಶ್ರೇಢಿಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಸಮಾಂತರ ಶ್ರೇಢಿಗಳ ಪರಿಕಲ್ಪನೆಯನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ಒಂದು ನಿರ್ದಿಷ್ಟ ಸಂಖ್ಯೆಯನ್ನು ಹಿಂದಿನ ಪದಕ್ಕೆ ಕೂಡಿಸುವುದರಿಂದ ಮುಂದಿನ ಪದವನ್ನು ಪಡೆಯುವ ಸಂಖ್ಯಾ ಪಟ್ಟಿಯನ್ನು ಸಮಾಂತರ ಶ್ರೇಢಿ ಎನ್ನಲಾಗುತ್ತದೆ. ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ನಾವು ಸಮಾಂತರ ಶ್ರೇಢಿಯ ಸಾಮಾನ್ಯ ರೂಪ, n-ನೇ ಪದವನ್ನು ಕಂಡುಹಿಡಿಯುವ ಸೂತ್ರ ಮತ್ತು ಮೊದಲ n ಪದಗಳ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯುವ ವಿಧಾನಗಳನ್ನು ಕಲಿಯುತ್ತೇವೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಸಮಾಂತರ ಶ್ರೇಢಿಯ n-ನೇ ಪದ: an = a + (n - 1)d
  • ಸಮಾಂತರ ಶ್ರೇಢಿಯ ಮೊದಲ n ಪದಗಳ ಮೊತ್ತ: Sn = n/2 [2a + (n - 1)d]
  • ಕೊನೆಯ ಪದ (l) ತಿಳಿದಿದ್ದಾಗ ಮೊತ್ತ: Sn = n/2 (a + l)
  • ಸಾಮಾನ್ಯ ವ್ಯತ್ಯಾಸ: d = ak+1 - ak

ಅಭ್ಯಾಸ 5.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಕೆಳಗಿನ ಸಂದರ್ಭಗಳಲ್ಲಿ ಯಾವುದು ಸಮಾಂತರ ಶ್ರೇಢಿಯನ್ನು ಉಂಟುಮಾಡುತ್ತದೆ ಮತ್ತು ಏಕೆ?

ಪರಿಹಾರ (i): ಪ್ರತಿ ಕಿ.ಮೀ ಗೆ ರೂ. 8 ಹೆಚ್ಚಾಗುವುದರಿಂದ, ಇದು ಸಮಾಂತರ ಶ್ರೇಢಿಯಾಗಿದೆ. (15, 23, 31, ...)

ಪರಿಹಾರ (ii): ಪ್ರತಿ ಬಾರಿ 1/4 ಭಾಗ ಹೊರಹೋಗುವುದರಿಂದ, ಉಳಿಯುವ ಪ್ರಮಾಣವು ಸ್ಥಿರವಾಗಿರುವುದಿಲ್ಲ. ಇದು ಸಮಾಂತರ ಶ್ರೇಢಿಯಲ್ಲ.

ಪರಿಹಾರ (iii): ಪ್ರತಿ ಮೀಟರ್‌ಗೆ ರೂ. 50 ಹೆಚ್ಚಾಗುವುದರಿಂದ, ಇದು ಸಮಾಂತರ ಶ್ರೇಢಿಯಾಗಿದೆ. (150, 200, 250, ...)

ಪರಿಹಾರ (iv): ಚಕ್ರಬಡ್ಡಿಯು ಪ್ರತಿ ವರ್ಷ ಬದಲಾಗುವುದರಿಂದ, ಇದು ಸಮಾಂತರ ಶ್ರೇಢಿಯಲ್ಲ.

ಪ್ರಶ್ನೆ 2: ಮೊದಲ ನಾಲ್ಕು ಪದಗಳನ್ನು ಬರೆಯಿರಿ (a=10, d=10):

ಪರಿಹಾರ: a1=10, a2=10+10=20, a3=20+10=30, a4=30+10=40. ಶ್ರೇಢಿ: 10, 20, 30, 40.

ಪ್ರಶ್ನೆ 3: ಮೊದಲ ಪದ ಮತ್ತು ಸಾಮಾನ್ಯ ವ್ಯತ್ಯಾಸ ಬರೆಯಿರಿ (3, 1, -1, -3):

ಪರಿಹಾರ: a = 3, d = 1 - 3 = -2.

ಪ್ರಶ್ನೆ 4: ಸಮಾಂತರ ಶ್ರೇಢಿಯಾಗಿದ್ದರೆ ಸಾಮಾನ್ಯ ವ್ಯತ್ಯಾಸ ಮತ್ತು ಮುಂದಿನ ಮೂರು ಪದಗಳನ್ನು ಬರೆಯಿರಿ (2, 4, 8, 16):

ಪರಿಹಾರ: 4-2=2, 8-4=4. ವ್ಯತ್ಯಾಸ ಸಮಾನವಿಲ್ಲ, ಆದ್ದರಿಂದ ಇದು ಸಮಾಂತರ ಶ್ರೇಢಿಯಲ್ಲ.

ಅಭ್ಯಾಸ 5.2 - ಪ್ರಮುಖ ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 4: 3, 8, 13, 18... ಶ್ರೇಢಿಯ ಎಷ್ಟನೇ ಪದ 78?

ಪರಿಹಾರ: a=3, d=5, an=78. ಸೂತ್ರ: 78 = 3 + (n-1)5 => 75 = (n-1)5 => 15 = n-1 => n = 16. 16ನೇ ಪದ.

ಪ್ರಶ್ನೆ 7: 11ನೇ ಪದ 38 ಮತ್ತು 16ನೇ ಪದ 73 ಆದರೆ 31ನೇ ಪದ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: a+10d=38, a+15d=73. ಕಳೆಯಲು: 5d=35 => d=7. a=38-70=-32. a31 = -32 + 30(7) = -32 + 210 = 178.

ಅಭ್ಯಾಸ 5.3 - ಪ್ರಮುಖ ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1(i): 2, 7, 12... ರ 10 ಪದಗಳ ಮೊತ್ತ.

ಪರಿಹಾರ: a=2, d=5, n=10. S10 = 10/2 [2(2) + 9(5)] = 5 [4 + 45] = 5 * 49 = 245.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಚಿಹ್ನೆಗಳ ಎಚ್ಚರಿಕೆ: ಸಾಮಾನ್ಯ ವ್ಯತ್ಯಾಸ (d) ಋಣಾತ್ಮಕವಾಗಿದ್ದಾಗ (ಉದಾ: -3), ಕಳೆಯುವಾಗ ಚಿಹ್ನೆಗಳನ್ನು ತಪ್ಪಿಸಬೇಡಿ.
  • n ನ ಮೌಲ್ಯ: ಪದಗಳ ಸಂಖ್ಯೆ (n) ಯಾವಾಗಲೂ ಧನಾತ್ಮಕ ಪೂರ್ಣಾಂಕವಾಗಿರಬೇಕು. ಅದು ಭಿನ್ನರಾಶಿ ಅಥವಾ ಋಣಾತ್ಮಕ ಬಂದರೆ, ಆ ಸಂಖ್ಯೆಯು ಆ ಶ್ರೇಢಿಯ ಪದವಲ್ಲ ಎಂದು ಅರ್ಥ.
  • ಸೂತ್ರದ ಆಯ್ಕೆ: ಕೊನೆಯ ಪದ (l) ಗೊತ್ತಿದ್ದಾಗ Sn = n/2(a+l) ಬಳಸುವುದು ಸಮಯ ಉಳಿಸುತ್ತದೆ.
