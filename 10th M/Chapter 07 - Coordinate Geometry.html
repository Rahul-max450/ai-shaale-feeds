ಅಧ್ಯಾಯ 7: ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತ (Coordinate Geometry)

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಸಮತಲದಲ್ಲಿರುವ ಬಿಂದುಗಳ ಸ್ಥಾನವನ್ನು ನಿರ್ಧರಿಸಲು ಬಳಸುವ ನಿರ್ದೇಶಾಂಕ ವ್ಯವಸ್ಥೆಯ ಬಗ್ಗೆ ತಿಳಿಸುತ್ತದೆ. ಎರಡು ಬಿಂದುಗಳ ನಡುವಿನ ಅಂತರವನ್ನು ಕಂಡುಹಿಡಿಯುವ 'ಅಂತರ ಸೂತ್ರ' (Distance Formula) ಮತ್ತು ಒಂದು ರೇಖಾಖಂಡವನ್ನು ನಿರ್ದಿಷ್ಟ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವ 'ವಿಭಾಗ ಸೂತ್ರ' (Section Formula) ಈ ಅಧ್ಯಾಯದ ಪ್ರಮುಖ ಭಾಗಗಳಾಗಿವೆ. ಇವುಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಜ್ಯಾಮಿತೀಯ ಆಕೃತಿಗಳ ಗುಣಲಕ್ಷಣಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲಾಗುತ್ತದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಅಂತರ ಸೂತ್ರ: P(x₁, y₁) ಮತ್ತು Q(x₂, y₂) ಬಿಂದುಗಳ ನಡುವಿನ ಅಂತರ PQ = √[(x₂ - x₁)² + (y₂ - y₁)²]
  • ಮೂಲ ಬಿಂದುವಿನಿಂದ ಅಂತರ: P(x, y) ಬಿಂದುವಿನಿಂದ ಮೂಲ ಬಿಂದುವಿನ (0, 0) ಅಂತರ OP = √(x² + y²)
  • ವಿಭಾಗ ಸೂತ್ರ (ಆಂತರಿಕ ವಿಭಾಗ): A(x₁, y₁) ಮತ್ತು B(x₂, y₂) ಬಿಂದುಗಳನ್ನು m₁ : m₂ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದು P(x, y) ನ ನಿರ್ದೇಶಾಂಕಗಳು:
    x = (m₁x₂ + m₂x₁) / (m₁ + m₂)
    y = (m₁y₂ + m₂y₁) / (m₁ + m₂)
  • ಮಧ್ಯಬಿಂದು ಸೂತ್ರ: A(x₁, y₁) ಮತ್ತು B(x₂, y₂) ಬಿಂದುಗಳ ಮಧ್ಯಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು = [(x₁ + x₂) / 2, (y₁ + y₂) / 2]

ಅಭ್ಯಾಸ 7.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಕೆಳಗಿನ ಬಿಂದುಗಳ ಜೋಡಿಗಳ ನಡುವಿನ ಅಂತರವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.

i) (2, 3), (4, 1)
ಪರಿಹಾರ: d = √[(4-2)² + (1-3)²] = √[2² + (-2)²] = √[4 + 4] = √8 = 2√2 ಮಾನಗಳು.

ii) (-5, 7), (-1, 3)
ಪರಿಹಾರ: d = √[(-1 - (-5))² + (3-7)²] = √[4² + (-4)²] = √[16 + 16] = √32 = 4√2 ಮಾನಗಳು.

iii) (a, b), (-a, -b)
ಪರಿಹಾರ: d = √[(-a - a)² + (-b - b)²] = √[(-2a)² + (-2b)²] = √[4a² + 4b²] = 2√(a² + b²) ಮಾನಗಳು.

ಪ್ರಶ್ನೆ 2: (0, 0) ಮತ್ತು (36, 15) ಬಿಂದುಗಳ ನಡುವಿನ ಅಂತರವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: d = √[(36-0)² + (15-0)²] = √[36² + 15²] = √[1296 + 225] = √1521 = 39 ಕಿ.ಮೀ.

ಪ್ರಶ್ನೆ 3: (1, 5), (2, 3) ಮತ್ತು (-2, -11) ಬಿಂದುಗಳು ಸರಳರೇಖಾಗತವೇ ಎಂದು ನಿರ್ಧರಿಸಿ.

ಪರಿಹಾರ: AB = √[(2-1)² + (3-5)²] = √5; BC = √[(-2-2)² + (-11-3)²] = √[16+196] = √212; AC = √[(-2-1)² + (-11-5)²] = √[9+256] = √265. AB + BC ≠ AC, ಆದ್ದರಿಂದ ಇವು ಸರಳರೇಖಾಗತವಲ್ಲ.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಅಂತರ ಸೂತ್ರದಲ್ಲಿ ಋಣಾತ್ಮಕ ಸಂಖ್ಯೆಗಳ ವರ್ಗ ಮಾಡುವಾಗ ಚಿಹ್ನೆಗಳ ಬಗ್ಗೆ ಎಚ್ಚರವಿರಲಿ (ಉದಾ: (-2)² = 4).
  • ವಿಭಾಗ ಸೂತ್ರದಲ್ಲಿ m₁ ಮತ್ತು m₂ ಅನುಪಾತವನ್ನು ಸರಿಯಾಗಿ ಗುರುತಿಸಿ.
  • ಜ್ಯಾಮಿತೀಯ ಆಕೃತಿಗಳನ್ನು (ಚೌಕ, ಆಯತ) ಸಾಬೀತುಪಡಿಸುವಾಗ ಕರ್ಣಗಳ ಉದ್ದವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಮರೆಯದಿರಿ.
