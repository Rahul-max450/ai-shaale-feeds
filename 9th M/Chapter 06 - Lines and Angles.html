ಅಧ್ಯಾಯ 06: ರೇಖೆಗಳು ಮತ್ತು ಕೋನಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ರೇಖೆಗಳು ಮತ್ತು ಕೋನಗಳ ಮೂಲಭೂತ ಗುಣಲಕ್ಷಣಗಳನ್ನು ವಿವರಿಸುತ್ತದೆ. ಎರಡು ಸರಳ ರೇಖೆಗಳು ಛೇದಿಸಿದಾಗ ಉಂಟಾಗುವ ಕೋನಗಳು, ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು, ರೇಖಾತ್ಮಕ ಜೋಡಿಗಳು ಮತ್ತು ಸಮಾಂತರ ರೇಖೆಗಳನ್ನು ಒಂದು ಛೇದಕವು ಛೇದಿಸಿದಾಗ ಉಂಟಾಗುವ ವಿವಿಧ ಕೋನಗಳ ಸಂಬಂಧಗಳನ್ನು ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಕಲಿಯುತ್ತೇವೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ರೇಖಾತ್ಮಕ ಜೋಡಿ ಆಧಾರ ಪ್ರಮೇಯ: ಒಂದು ಸರಳ ರೇಖೆಯ ಮೇಲೆ ಒಂದು ಕಿರಣವು ನಿಂತಾಗ ಉಂಟಾಗುವ ಪಾರ್ಶ್ವ ಕೋನಗಳ ಮೊತ್ತ 180°.
  • ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು: ಎರಡು ಸರಳ ರೇಖೆಗಳು ಪರಸ್ಪರ ಛೇದಿಸಿದಾಗ ಉಂಟಾಗುವ ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು ಯಾವಾಗಲೂ ಸಮನಾಗಿರುತ್ತವೆ.
  • ಸಮಾಂತರ ರೇಖೆಗಳ ಗುಣ: ಒಂದೇ ರೇಖೆಗೆ ಸಮಾಂತರವಾಗಿರುವ ರೇಖೆಗಳು ಪರಸ್ಪರ ಸಮಾಂತರವಾಗಿರುತ್ತವೆ.
  • ಪಾರ್ಶ್ವ ಕೋನಗಳು: ಒಂದು ಸಾಮಾನ್ಯ ಶೃಂಗ ಮತ್ತು ಸಾಮಾನ್ಯ ಬಾಹುವನ್ನು ಹೊಂದಿರುವ ಕೋನಗಳು.

ಅಭ್ಯಾಸ 6.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಚಿತ್ರ 6.13 ರಲ್ಲಿ AB ಮತ್ತು CD ಸರಳ ರೇಖೆಗಳು O ಬಿಂದುವಿನಲ್ಲಿ ಛೇದಿಸುತ್ತವೆ. ∠AOC + ∠BOE = 70° ಮತ್ತು ∠BOD = 40° ಆದರೆ ∠BOE ಮತ್ತು ಸರಳಾಧಿಕ ∠COE ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
1. AB ಒಂದು ಸರಳ ರೇಖೆ, ಆದ್ದರಿಂದ ∠AOC + ∠COE + ∠BOE = 180°
2. (∠AOC + ∠BOE) + ∠COE = 180°
3. 70° + ∠COE = 180° => ∠COE = 110°
4. ∠BOD = 40° (ದತ್ತ). ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳ ನಿಯಮದಂತೆ ∠AOC = ∠BOD = 40°
5. ∠AOC + ∠BOE = 70° => 40° + ∠BOE = 70° => ∠BOE = 30°
6. ಸರಳಾಧಿಕ ∠COE = 360° - 110° = 250°

ಪ್ರಶ್ನೆ 2: ಚಿತ್ರ 6.14 ರಲ್ಲಿ XY ಮತ್ತು MN ಸರಳ ರೇಖೆಗಳು O ಬಿಂದುವಿನಲ್ಲಿ ಛೇದಿಸುತ್ತವೆ. ∠POY = 90° ಮತ್ತು a : b = 2 : 3 ಆದರೆ c ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
1. a + b + ∠POY = 180° (ರೇಖಾತ್ಮಕ ಜೋಡಿ)
2. a + b + 90° = 180° => a + b = 90°
3. a = 2x, b = 3x ಎಂದು ಭಾವಿಸಿದರೆ: 2x + 3x = 90° => 5x = 90° => x = 18°
4. a = 36°, b = 54°
5. MN ಒಂದು ಸರಳ ರೇಖೆ, ಆದ್ದರಿಂದ b + c = 180°
6. 54° + c = 180° => c = 126°

ಪ್ರಶ್ನೆ 3: ಚಿತ್ರ 6.15 ರಲ್ಲಿ ∠PQR = ∠PRQ ಆದರೆ ∠PQS = ∠PRT ಎಂದು ಸಾಧಿಸಿ.

ಪರಿಹಾರ:
1. ∠PQS + ∠PQR = 180° (ರೇಖಾತ್ಮಕ ಜೋಡಿ)
2. ∠PRT + ∠PRQ = 180° (ರೇಖಾತ್ಮಕ ಜೋಡಿ)
3. ಆದ್ದರಿಂದ, ∠PQS + ∠PQR = ∠PRT + ∠PRQ
4. ದತ್ತದಂತೆ ∠PQR = ∠PRQ, ಆದ್ದರಿಂದ ಇವೆರಡನ್ನು ರದ್ದುಗೊಳಿಸಿದರೆ:
5. ∠PQS = ∠PRT (ಸಾಧಿತವಾಯಿತು).

ಪ್ರಶ್ನೆ 4: ಚಿತ್ರ 6.16 ರಲ್ಲಿ x + y = w + z ಆದರೆ AOB ಒಂದು ಸರಳ ರೇಖೆ ಎಂದು ಸಾಧಿಸಿ.

ಪರಿಹಾರ:
1. ಒಂದು ಬಿಂದುವಿನ ಸುತ್ತಲಿನ ಕೋನಗಳ ಮೊತ್ತ 360°: x + y + w + z = 360°
2. ದತ್ತದಂತೆ x + y = w + z, ಆದ್ದರಿಂದ (x + y) + (x + y) = 360°
3. 2(x + y) = 360° => x + y = 180°
4. x ಮತ್ತು y ರೇಖಾತ್ಮಕ ಜೋಡಿಯಾಗಿರುವುದರಿಂದ, AOB ಒಂದು ಸರಳ ರೇಖೆಯಾಗಿದೆ.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಯಾವಾಗಲೂ ರೇಖಾತ್ಮಕ ಜೋಡಿಗಳನ್ನು ಗುರುತಿಸಲು ಪ್ರಯತ್ನಿಸಿ, ಇದು ಹೆಚ್ಚಿನ ಲೆಕ್ಕಗಳಲ್ಲಿ ಸಹಾಯ ಮಾಡುತ್ತದೆ.
  • ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು ಸಮನಾಗಿರುತ್ತವೆ ಎಂಬ ನಿಯಮವನ್ನು ಮರೆಯಬೇಡಿ.
  • ಸಮಾಂತರ ರೇಖೆಗಳ ಲೆಕ್ಕ ಮಾಡುವಾಗ ಛೇದಕ ರೇಖೆಯನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಗುರುತಿಸಿ.
