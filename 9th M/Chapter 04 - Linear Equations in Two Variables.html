ಅಧ್ಯಾಯ 4: ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳು

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಎರಡು ಚರಾಕ್ಷರಗಳನ್ನು (x ಮತ್ತು y) ಒಳಗೊಂಡಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಪರಿಚಯವನ್ನು ನೀಡುತ್ತದೆ. ಇವುಗಳನ್ನು ax + by + c = 0 ರೂಪದಲ್ಲಿ ವ್ಯಕ್ತಪಡಿಸಲಾಗುತ್ತದೆ. ಈ ಸಮೀಕರಣಗಳ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು, ಅವುಗಳನ್ನು ಸಮೀಕರಣದ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು ಮತ್ತು ಇವುಗಳಿಗೆ ಅಪರಿಮಿತ ಪರಿಹಾರಗಳಿರುತ್ತವೆ ಎಂಬುದನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು ಈ ಅಧ್ಯಾಯದ ಮುಖ್ಯ ಉದ್ದೇಶವಾಗಿದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಸಾಮಾನ್ಯ ರೂಪ: ax + by + c = 0 (ಇಲ್ಲಿ a, b ಮತ್ತು c ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು, ಮತ್ತು a, b ಎರಡೂ ಏಕಕಾಲದಲ್ಲಿ ಸೊನ್ನೆಯಾಗಿರಬಾರದು).
  • ಪರಿಹಾರ: (x, y) ಜೋಡಿಯು ಸಮೀಕರಣವನ್ನು ತೃಪ್ತಿಪಡಿಸಿದರೆ, ಅದನ್ನು ಪರಿಹಾರ ಎನ್ನಲಾಗುತ್ತದೆ.
  • ಪರಿಹಾರಗಳ ಸಂಖ್ಯೆ: ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಕ್ಕೆ ಅಪರಿಮಿತ ಪರಿಹಾರಗಳಿರುತ್ತವೆ.

ಅಭ್ಯಾಸ 4.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಒಂದು ನೋಟ್ ಪುಸ್ತಕದ ಬೆಲೆಯು ಒಂದು ಪೆನ್ನಿನ ಬೆಲೆಯ ಎರಡರಷ್ಟಿದೆ. ಈ ಹೇಳಿಕೆಯನ್ನು ಪ್ರತಿನಿಧಿಸಲು ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣವನ್ನು ಬರೆಯಿರಿ.

ಪರಿಹಾರ: ನೋಟ್ ಪುಸ್ತಕದ ಬೆಲೆ = x, ಪೆನ್ನಿನ ಬೆಲೆ = y ಎಂದು ತೆಗೆದುಕೊಳ್ಳೋಣ.
ಪ್ರಶ್ನೆಯ ಪ್ರಕಾರ, x = 2y
ಅಂದರೆ, x - 2y = 0

ಪ್ರಶ್ನೆ 2: ಕೆಳಗಿನ ಸಮೀಕರಣಗಳನ್ನು ax + by + c = 0 ರೂಪದಲ್ಲಿ ಬರೆದು, a, b ಮತ್ತು c ಬೆಲೆಗಳನ್ನು ಬರೆಯಿರಿ.

  1. 2x + 3y = 9.35
    ರೂಪ: 2x + 3y - 9.35 = 0. ಇಲ್ಲಿ a=2, b=3, c=-9.35
  2. x - y/5 - 10 = 0
    ರೂಪ: 1x - (1/5)y - 10 = 0. ಇಲ್ಲಿ a=1, b=-1/5, c=-10
  3. -2x + 3y = 6
    ರೂಪ: -2x + 3y - 6 = 0. ಇಲ್ಲಿ a=-2, b=3, c=-6
  4. x = 3y
    ರೂಪ: 1x - 3y + 0 = 0. ಇಲ್ಲಿ a=1, b=-3, c=0
  5. 2x = -5y
    ರೂಪ: 2x + 5y + 0 = 0. ಇಲ್ಲಿ a=2, b=5, c=0
  6. 3x + 2 = 0
    ರೂಪ: 3x + 0y + 2 = 0. ಇಲ್ಲಿ a=3, b=0, c=2
  7. y - 2 = 0
    ರೂಪ: 0x + 1y - 2 = 0. ಇಲ್ಲಿ a=0, b=1, c=-2
  8. 5 = 2x
    ರೂಪ: -2x + 0y + 5 = 0. ಇಲ್ಲಿ a=-2, b=0, c=5

ಅಭ್ಯಾಸ 4.2 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: y = 3x + 5 ಸಮೀಕರಣಕ್ಕೆ ಯಾವುದು ಸರಿ? (ಉತ್ತರ: iii) ಅಪರಿಮಿತ ಪರಿಹಾರಗಳಿವೆ.

ಪ್ರಶ್ನೆ 2: ಕೆಳಗಿನ ಸಮೀಕರಣಗಳಿಗೆ ನಾಲ್ಕು ಪರಿಹಾರಗಳನ್ನು ಬರೆಯಿರಿ.

  1. 2x + y = 7: (0,7), (1,5), (2,3), (3,1)
  2. πx + y = 9: (0,9), (1, 9-π), (2, 9-2π), (-1, 9+π)
  3. x = 4y: (0,0), (4,1), (8,2), (-4,-1)

ಪ್ರಶ್ನೆ 3: x - 2y = 4 ಸಮೀಕರಣಕ್ಕೆ ಪರಿಹಾರಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ.

  1. (0,2): 0 - 2(2) = -4 (ಅಲ್ಲ)
  2. (2,0): 2 - 2(0) = 2 (ಅಲ್ಲ)
  3. (4,0): 4 - 2(0) = 4 (ಹೌದು)
  4. (√2, 4√2): √2 - 8√2 = -7√2 (ಅಲ್ಲ)
  5. (1,1): 1 - 2(1) = -1 (ಅಲ್ಲ)

ಪ್ರಶ್ನೆ 4: 2x + 3y = k ಸಮೀಕರಣಕ್ಕೆ x=2, y=1 ಪರಿಹಾರವಾದರೆ k ಬೆಲೆ ಎಷ್ಟು?

ಪರಿಹಾರ: 2(2) + 3(1) = k
4 + 3 = k
k = 7

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಸಮೀಕರಣವನ್ನು ax + by + c = 0 ರೂಪಕ್ಕೆ ತರುವಾಗ ಚಿಹ್ನೆಗಳನ್ನು (positive/negative) ಎಚ್ಚರಿಕೆಯಿಂದ ಗಮನಿಸಿ.
  • ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವಾಗ, x ಗೆ 0 ಅಥವಾ 1 ರಂತಹ ಸಣ್ಣ ಬೆಲೆಗಳನ್ನು ನೀಡಿ y ಅನ್ನು ಸುಲಭವಾಗಿ ಕಂಡುಹಿಡಿಯಬಹುದು.
  • ಯಾವುದೇ ಚರಾಕ್ಷರ ಇಲ್ಲದಿದ್ದರೆ, ಅದರ ಸಹಗುಣಕವನ್ನು 0 ಎಂದು ಪರಿಗಣಿಸಿ (ಉದಾ: 0x ಅಥವಾ 0y).
