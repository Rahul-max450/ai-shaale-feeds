ಅಧ್ಯಾಯ 2: ಬಹುಪದೋಕ್ತಿಗಳು (Polynomials)

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಬಹುಪದೋಕ್ತಿಗಳ ಪರಿಚಯ, ಅವುಗಳ ವಿಧಗಳು (ಏಕಪದೋಕ್ತಿ, ದ್ವಿಪದೋಕ್ತಿ, ತ್ರಿಪದೋಕ್ತಿ), ಬಹುಪದೋಕ್ತಿಯ ಘಾತ (Degree), ಶೂನ್ಯತೆಗಳು (Zeros), ಶೇಷ ಪ್ರಮೇಯ ಮತ್ತು ಅಪವರ್ತನ ಪ್ರಮೇಯಗಳ ಬಗ್ಗೆ ವಿವರಿಸುತ್ತದೆ. ಅಲ್ಲದೆ, ಬೀಜಗಣಿತದ ನಿತ್ಯಸಮೀಕರಣಗಳನ್ನು ಬಳಸಿ ಬಹುಪದೋಕ್ತಿಗಳನ್ನು ಅಪವರ್ತಿಸುವುದು ಮತ್ತು ವಿಸ್ತರಿಸುವುದನ್ನು ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಕಲಿಯುತ್ತೇವೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • (x + y)² = x² + 2xy + y²
  • (x – y)² = x² – 2xy + y²
  • x² – y² = (x + y)(x – y)
  • (x + a)(x + b) = x² + (a + b)x + ab
  • (x + y + z)² = x² + y² + z² + 2xy + 2yz + 2zx
  • (x + y)³ = x³ + y³ + 3xy(x + y)
  • (x – y)³ = x³ – y³ – 3xy(x – y)
  • x³ + y³ + z³ – 3xyz = (x + y + z)(x² + y² + z² – xy – yz – zx)

ಅಭ್ಯಾಸ 2.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಕೆಳಗಿನವುಗಳಲ್ಲಿ ಯಾವುವು ಒಂದು ಚರಾಕ್ಷರವುಳ್ಳ ಬಹುಪದೋಕ್ತಿಗಳು?

ಪರಿಹಾರ:

  • (i) 4x² – 3x + 7: ಹೌದು, ಇದು ಒಂದು ಚರಾಕ್ಷರವುಳ್ಳ ಬಹುಪದೋಕ್ತಿ.
  • (ii) y² + √2: ಹೌದು, ಇದು ಒಂದು ಚರಾಕ್ಷರವುಳ್ಳ ಬಹುಪದೋಕ್ತಿ.
  • (iii) 3√t + t√2: ಅಲ್ಲ, ಏಕೆಂದರೆ t ನ ಘಾತವು ಪೂರ್ಣಸಂಖ್ಯೆಯಲ್ಲ (1/2).
  • (iv) y + 2/y: ಅಲ್ಲ, ಏಕೆಂದರೆ y ನ ಘಾತವು ಋಣಾತ್ಮಕವಾಗಿದೆ.
  • (v) x¹⁰ + y³ + t⁵⁰: ಅಲ್ಲ, ಇದು ಮೂರು ಚರಾಕ್ಷರಗಳನ್ನು ಹೊಂದಿದೆ.

ಪ್ರಶ್ನೆ 2: x² ನ ಸಹಗುಣಕಗಳನ್ನು ಬರೆಯಿರಿ.

ಪರಿಹಾರ:

  • (i) 2 + x² + x: ಸಹಗುಣಕ = 1
  • (ii) 2 – x² + x³: ಸಹಗುಣಕ = -1
  • (iii) (π/2)x² + x: ಸಹಗುಣಕ = π/2
  • (iv) √2x – 1: ಸಹಗುಣಕ = 0 (x² ಪದವಿಲ್ಲ)

ಪ್ರಶ್ನೆ 4: ಬಹುಪದೋಕ್ತಿಯ ಡಿಗ್ರಿಯನ್ನು ಬರೆಯಿರಿ.

ಪರಿಹಾರ:

  • (i) 5x³ + 4x² + 7x: ಡಿಗ್ರಿ = 3
  • (ii) 4 – y²: ಡಿಗ್ರಿ = 2
  • (iii) 5t – √7: ಡಿಗ್ರಿ = 1
  • (iv) 3: ಡಿಗ್ರಿ = 0

ಅಭ್ಯಾಸ 2.2 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: p(x) = 5x – 4x² + 3 ರ ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:

  • (i) x = 0: 5(0) – 4(0)² + 3 = 3
  • (ii) x = -1: 5(-1) – 4(-1)² + 3 = -5 – 4 + 3 = -6
  • (iii) x = 2: 5(2) – 4(2)² + 3 = 10 – 16 + 3 = -3

ಅಭ್ಯಾಸ 2.3 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 4: ಅಪವರ್ತಿಸಿ (Factorize):

ಪರಿಹಾರ (i): 12x² – 7x + 1

ಮಧ್ಯದ ಪದವನ್ನು ವಿಭಜಿಸಿ: 12x² – 4x – 3x + 1

= 4x(3x – 1) – 1(3x – 1) = (3x – 1)(4x – 1)

ಪರಿಹಾರ (ii): 2x² + 7x + 3

= 2x² + 6x + 1x + 3

= 2x(x + 3) + 1(x + 3) = (x + 3)(2x + 1)

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಘಾತದ ನಿಯಮ: ಬಹುಪದೋಕ್ತಿಯಲ್ಲಿ ಚರಾಕ್ಷರದ ಘಾತವು ಯಾವಾಗಲೂ ಧನಾತ್ಮಕ ಪೂರ್ಣಸಂಖ್ಯೆಯಾಗಿರಬೇಕು. ಋಣಾತ್ಮಕ ಅಥವಾ ಭಿನ್ನರಾಶಿ ಘಾತವಿದ್ದರೆ ಅದು ಬಹುಪದೋಕ್ತಿಯಲ್ಲ.
  • ಸಹಗುಣಕ: x² ನ ಸಹಗುಣಕ ಕೇಳಿದಾಗ, x² ಪದವೇ ಇಲ್ಲದಿದ್ದರೆ ಅದರ ಸಹಗುಣಕ 0 ಎಂದು ಬರೆಯಿರಿ.
  • ಅಪವರ್ತನ: ಮಧ್ಯದ ಪದವನ್ನು ವಿಭಜಿಸುವಾಗ, ಗುಣಲಬ್ಧವು ಮೊದಲ ಮತ್ತು ಕೊನೆಯ ಪದಗಳ ಗುಣಲಬ್ಧಕ್ಕೆ ಸಮನಾಗಿರಬೇಕು ಎಂಬುದನ್ನು ಮರೆಯದಿರಿ.
  • ಚಿಹ್ನೆಗಳ ಎಚ್ಚರಿಕೆ: (x – y)² ಮತ್ತು (x + y)² ಸೂತ್ರಗಳನ್ನು ಬಳಸುವಾಗ ಚಿಹ್ನೆಗಳ (+/-) ಬಗ್ಗೆ ಜಾಗರೂಕರಾಗಿರಿ.
