ಅಧ್ಯಾಯ 10 - ಹೆರಾನ್‌ನ ಸೂತ್ರ (Heron's Formula)

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ತ್ರಿಭುಜದ ಎತ್ತರವು ತಿಳಿದಿಲ್ಲದಿದ್ದಾಗ, ಕೇವಲ ಅದರ ಮೂರು ಬಾಹುಗಳ ಅಳತೆಗಳನ್ನು ಬಳಸಿಕೊಂಡು ತ್ರಿಭುಜದ ವಿಸ್ತೀರ್ಣವನ್ನು ಕಂಡುಹಿಡಿಯುವ 'ಹೆರಾನ್‌ನ ಸೂತ್ರ'ದ ಬಗ್ಗೆ ವಿವರಿಸುತ್ತದೆ. ಇದು ಅಸಮಬಾಹು ತ್ರಿಭುಜಗಳ ವಿಸ್ತೀರ್ಣವನ್ನು ಲೆಕ್ಕಾಚಾರ ಮಾಡಲು ಅತ್ಯಂತ ಉಪಯುಕ್ತವಾದ ವಿಧಾನವಾಗಿದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಅರ್ಧ ಸುತ್ತಳತೆ (s): s = (a + b + c) / 2
  • ಹೆರಾನ್‌ನ ಸೂತ್ರ (ತ್ರಿಭುಜದ ವಿಸ್ತೀರ್ಣ): √[s(s - a)(s - b)(s - c)]
  • ಇಲ್ಲಿ a, b ಮತ್ತು c ತ್ರಿಭುಜದ ಮೂರು ಬಾಹುಗಳಾಗಿವೆ.

ಅಭ್ಯಾಸ 10.1 - ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: "ಮುಂದೆ ಶಾಲೆ ಇದೆ" ಎಂದು ಸೂಚಿಸುವ ಸಂಚಾರ ಸಂಜ್ಞಾಪಲಕವು ಸಮಬಾಹು ತ್ರಿಭುಜಾಕಾರದಲ್ಲಿದ್ದು ಅದರ ಬಾಹು 'a' ಆಗಿದೆ. ಹೆರಾನ್‌ನ ಸೂತ್ರ ಬಳಸಿ ವಿಸ್ತೀರ್ಣ ಕಂಡುಹಿಡಿಯಿರಿ. ಸುತ್ತಳತೆ 180 cm ಇದ್ದರೆ ವಿಸ್ತೀರ್ಣ ಎಷ್ಟು?

ಪರಿಹಾರ:
ಸಮಬಾಹು ತ್ರಿಭುಜದ ಬಾಹುಗಳು a, a, a.
s = (a + a + a) / 2 = 3a / 2
ವಿಸ್ತೀರ್ಣ = √[s(s-a)(s-a)(s-a)] = √[(3a/2)(a/2)(a/2)(a/2)] = (√3 / 4) * a²
ಸುತ್ತಳತೆ = 3a = 180 cm => a = 60 cm.
ವಿಸ್ತೀರ್ಣ = (√3 / 4) * 60 * 60 = 900√3 cm².

ಪ್ರಶ್ನೆ 2: ಮೇಲ್ಸೇತುವೆಯ ತ್ರಿಭುಜಾಕಾರದ ಗೋಡೆಯ ಬಾಹುಗಳು 122 m, 22 m ಮತ್ತು 120 m. ಜಾಹೀರಾತು ವರ್ಷಕ್ಕೆ ಪ್ರತಿ m² ಗೆ ₹ 5000 ನೀಡುತ್ತದೆ. 3 ತಿಂಗಳಿಗೆ ಬಾಡಿಗೆ ಎಷ್ಟು?

ಪರಿಹಾರ:
s = (122 + 22 + 120) / 2 = 264 / 2 = 132 m.
ವಿಸ್ತೀರ್ಣ = √[132(132-122)(132-22)(132-120)] = √[132 * 10 * 110 * 12] = 1320 m².
ಒಂದು ವರ್ಷದ ಬಾಡಿಗೆ = 1320 * 5000 = ₹ 66,00,000.
3 ತಿಂಗಳ ಬಾಡಿಗೆ = 66,00,000 * (3/12) = ₹ 16,50,000.

ಪ್ರಶ್ನೆ 3: ಗೋಡೆಯ ಬಾಹುಗಳು 15m, 11m ಮತ್ತು 6m. ಬಣ್ಣ ಬಳಿದ ಜಾಗದ ವಿಸ್ತೀರ್ಣ ಎಷ್ಟು?

ಪರಿಹಾರ:
s = (15 + 11 + 6) / 2 = 32 / 2 = 16 m.
ವಿಸ್ತೀರ್ಣ = √[16(16-15)(16-11)(16-6)] = √[16 * 1 * 5 * 10] = √800 = 20√2 m².

ಪ್ರಶ್ನೆ 4: ಎರಡು ಬಾಹುಗಳು 18cm ಮತ್ತು 10cm, ಸುತ್ತಳತೆ 42cm ಇರುವ ತ್ರಿಭುಜದ ವಿಸ್ತೀರ್ಣ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
ಮೂರನೇ ಬಾಹು c = 42 - (18 + 10) = 14 cm.
s = 42 / 2 = 21 cm.
ವಿಸ್ತೀರ್ಣ = √[21(21-18)(21-10)(21-14)] = √[21 * 3 * 11 * 7] = √441 * 3 = 21√3 cm².

ಪ್ರಶ್ನೆ 5: ಬಾಹುಗಳ ಅನುಪಾತ 12:17:25 ಮತ್ತು ಸುತ್ತಳತೆ 540cm. ವಿಸ್ತೀರ್ಣ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
ಬಾಹುಗಳು 12x, 17x, 25x. 12x+17x+25x = 54x = 540 => x = 10.
ಬಾಹುಗಳು: 120, 170, 250. s = 540/2 = 270.
ವಿಸ್ತೀರ್ಣ = √[270(270-120)(270-170)(270-250)] = √[270 * 150 * 100 * 20] = 9000 cm².

ಪ್ರಶ್ನೆ 6: ಸಮದ್ವಿಬಾಹು ತ್ರಿಭುಜದ ಸುತ್ತಳತೆ 30cm ಮತ್ತು ಸಮಬಾಹುಗಳ ಉದ್ದ 12cm. ವಿಸ್ತೀರ್ಣ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ:
ಬಾಹುಗಳು 12, 12, c. 12+12+c = 30 => c = 6 cm.
s = 30/2 = 15 cm.
ವಿಸ್ತೀರ್ಣ = √[15(15-12)(15-12)(15-6)] = √[15 * 3 * 3 * 9] = 9√15 cm².

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಲೆಕ್ಕಾಚಾರ ಮಾಡುವಾಗ 's' (ಅರ್ಧ ಸುತ್ತಳತೆ) ಕಂಡುಹಿಡಿಯಲು ಮರೆಯದಿರಿ.
  • ವರ್ಗಮೂಲದ (Square root) ಒಳಗಿನ ಸಂಖ್ಯೆಗಳನ್ನು ಗುಣಿಸುವ ಬದಲು, ಅವಿಭಾಜ್ಯ ಅಪವರ್ತನಗಳಾಗಿ (Prime factors) ವಿಂಗಡಿಸಿ ಹೊರತೆಗೆಯುವುದು ಸುಲಭ.
  • ತ್ರಿಭುಜದ ಮೂರು ಬಾಹುಗಳ ಮೊತ್ತವು ಸುತ್ತಳತೆಗೆ ಸಮನಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ.
