ಅಧ್ಯಾಯ 09 - ಬೌಧಾಯನ ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯ

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಪ್ರಾಚೀನ ಭಾರತೀಯ ಗಣಿತಜ್ಞ ಬೌಧಾಯನರು ಪ್ರತಿಪಾದಿಸಿದ ಪ್ರಮೇಯದ ಬಗ್ಗೆ ತಿಳಿಸುತ್ತದೆ. ಲಂಬಕೋನ ತ್ರಿಭುಜದ ಬಾಹುಗಳ ನಡುವಿನ ಸಂಬಂಧ, ಚೌಕಗಳ ವಿಸ್ತೀರ್ಣವನ್ನು ದ್ವಿಗುಣಗೊಳಿಸುವಿಕೆ, ಮತ್ತು ಪೈಥಾಗೊರಸ್ ತ್ರಿವಳಿಗಳ (Pythagorean Triples) ರಚನೆಯನ್ನು ಈ ಅಧ್ಯಾಯವು ವಿವರಿಸುತ್ತದೆ. ಇದು ಜ್ಯಾಮಿತಿಯ ಮೂಲಭೂತ ತತ್ವಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಬೌಧಾಯನ-ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯ: ಲಂಬಕೋನ ತ್ರಿಭುಜದಲ್ಲಿ, ಲಂಬಕೋನಕ್ಕೆ ಅಭಿಮುಖವಾಗಿರುವ ಬಾಹು (ವಿಕರ್ಣ 'c') ಮತ್ತು ಉಳಿದೆರಡು ಬಾಹುಗಳು ('a' ಮತ್ತು 'b') ಆಗಿದ್ದರೆ: a² + b² = c²
  • ಬೌಧಾಯನ ತ್ರಿವಳಿಗಳು: (a, b, c) ತ್ರಿವಳಿಗಳು a² + b² = c² ಅನ್ನು ಪೂರೈಸಿದರೆ, ಅವು ಬೌಧಾಯನ ತ್ರಿವಳಿಗಳು.
  • ತ್ರಿವಳಿಗಳ ಗುಣಾಕಾರ: (a, b, c) ಒಂದು ತ್ರಿವಳಿಯಾದರೆ, (ka, kb, kc) ಕೂಡ ಒಂದು ತ್ರಿವಳಿಯಾಗುತ್ತದೆ (ಇಲ್ಲಿ k ಒಂದು ಧನ ಪೂರ್ಣಾಂಕ).

ಅಭ್ಯಾಸದ ಪ್ರಶ್ನೆಗಳು ಮತ್ತು ಪರಿಹಾರಗಳು

ಪುಟ 47 ರ ಪ್ರಶ್ನೆಗಳು:

ಪ್ರಶ್ನೆ 1: ಎರಡು ಚೌಕಗಳ ವಿಸ್ತೀರ್ಣದ ದ್ವಿಗುಣದ ಚೌಕವನ್ನು ರಚಿಸುವ ವಿಧಾನ.

ಪರಿಹಾರ: ಎರಡು ಸಮಾನ ಚೌಕಗಳನ್ನು ತೆಗೆದುಕೊಂಡು, ಅವುಗಳನ್ನು ತ್ರಿಭುಜಗಳಾಗಿ ವಿಭಜಿಸಿ, ವಿಕರ್ಣದ ಮೇಲೆ ಹೊಸ ಚೌಕವನ್ನು ರಚಿಸುವುದರಿಂದ ಮೂಲ ಚೌಕದ ವಿಸ್ತೀರ್ಣದ ಎರಡು ಪಟ್ಟು ವಿಸ್ತೀರ್ಣವನ್ನು ಪಡೆಯಬಹುದು.

ಪ್ರಶ್ನೆ 2: ಸಮದ್ವಿಬಾಹು ಲಂಬಕೋನ ತ್ರಿಭುಜದ ಎರಡು ಸಮಬಾಹುಗಳ ಉದ್ದ ನೀಡಲಾಗಿದೆ. ವಿಕರ್ಣದ ಉದ್ದ ಕಂಡುಹಿಡಿಯಿರಿ.

ಪರಿಹಾರ: ಸೂತ್ರ c² = a² + a² = 2a² => c = a√2

  • (a) 3: c = 3√2 ≈ 4.24
  • (b) 4: c = 4√2 ≈ 5.65
  • (c) 6: c = 6√2 ≈ 8.48
  • (d) 8: c = 8√2 ≈ 11.31
  • (e) 9: c = 9√2 ≈ 12.72

ಪ್ರಶ್ನೆ 3: ವಿಕರ್ಣ 10cm ಇದ್ದರೆ, ಇತರ ಬಾಹುಗಳು ಎಷ್ಟು?

ಪರಿಹಾರ: c² = 2a² => 10² = 2a² => 100 = 2a² => a² = 50 => a = √50 ≈ 7.07 cm.

ಪುಟ 55 ರ ಪ್ರಶ್ನೆಗಳು:

ಪ್ರಶ್ನೆ 1: ಬಾಹುಗಳು 5cm ಮತ್ತು 12cm ಇದ್ದರೆ ವಿಕರ್ಣ ಎಷ್ಟು?

ಪರಿಹಾರ: c² = 5² + 12² = 25 + 144 = 169. c = √169 = 13 cm.

ಪ್ರಶ್ನೆ 2: ಬಾಹು 8cm ಮತ್ತು ವಿಕರ್ಣ 17cm ಇದ್ದರೆ ಮೂರನೇ ಬಾಹು ಎಷ್ಟು?

ಪರಿಹಾರ: 8² + b² = 17² => 64 + b² = 289 => b² = 225 => b = 15 cm.

ಪ್ರಶ್ನೆ 4: ಬಾಹುಗಳ ಉದ್ದ ಕಂಡುಹಿಡಿಯಿರಿ:

  • i) a=5, b=7 => c = √(25+49) = √74 ≈ 8.6
  • ii) a=8, b=12 => c = √(64+144) = √208 ≈ 14.4
  • iii) a=9, c=15 => b = √(225-81) = √144 = 12
  • iv) a=7, b=12 => c = √(49+144) = √193 ≈ 13.89
  • v) a=1.5, b=3.5 => c = √(2.25+12.25) = √14.5 ≈ 3.8

ಪುಟ 62 ರ ಪ್ರಶ್ನೆಗಳು:

ಪ್ರಶ್ನೆ 1: 5cm ಬಾಹುವಿನ ಚೌಕದ ವಿಕರ್ಣ?

ಪರಿಹಾರ: c = 5√2 ≈ 7.07 cm.

ಪ್ರಶ್ನೆ 2: ಬಾಹುಗಳ ಉದ್ದ (ಚಿತ್ರದ ಆಧಾರದ ಮೇಲೆ):

  • 7, 9 => c = √(49+81) = √130 ≈ 11.4
  • 4, 10 => c = √(16+100) = √116 ≈ 10.77
  • 40, 41 => b = √(41² - 40²) = √(1681-1600) = √81 = 9
  • 27, 45 => b = √(45² - 27²) = √(2025-729) = √1296 = 36

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ವಿಕರ್ಣವು ಯಾವಾಗಲೂ ಲಂಬಕೋನ ತ್ರಿಭುಜದ ಅತಿ ಉದ್ದದ ಬಾಹುವಾಗಿರುತ್ತದೆ. ಲೆಕ್ಕ ಮಾಡುವಾಗ ವಿಕರ್ಣವನ್ನು 'c' ಎಂದು ಗುರುತಿಸಲು ಮರೆಯದಿರಿ.
  • ವರ್ಗಮೂಲ (Square root) ಕಂಡುಹಿಡಿಯುವಾಗ ದಶಮಾಂಶ ಸ್ಥಾನಗಳನ್ನು ಎಚ್ಚರಿಕೆಯಿಂದ ಲೆಕ್ಕಿಸಿ.
  • ಬೌಧಾಯನ ತ್ರಿವಳಿಗಳನ್ನು (3, 4, 5), (5, 12, 13) ಕಂಠಪಾಠ ಮಾಡಿಕೊಂಡರೆ ಪರೀಕ್ಷೆಯಲ್ಲಿ ಲೆಕ್ಕಗಳನ್ನು ವೇಗವಾಗಿ ಬಿಡಿಸಬಹುದು.
