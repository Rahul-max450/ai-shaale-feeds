ಅಧ್ಯಾಯ 07: ಸಮಾನುಪಾತತೆಯ ತಾರ್ಕಿಕತೆ

ಅಧ್ಯಾಯದ ಸಂಕ್ಷಿಪ್ತ ಮಾಹಿತಿ

ಈ ಅಧ್ಯಾಯವು ಸಮಾನುಪಾತದ ಪರಿಕಲ್ಪನೆಯನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ಎರಡು ಪ್ರಮಾಣಗಳ ನಡುವಿನ ಸಂಬಂಧವನ್ನು ಹೇಗೆ ಹೋಲಿಸುವುದು, ಅವುಗಳ ನಡುವಿನ ಅನುಪಾತಗಳನ್ನು ಹೇಗೆ ಸರಳೀಕರಿಸುವುದು ಮತ್ತು ಸಮಾನುಪಾತದ ನಿಯಮಗಳನ್ನು (ತ್ರೈರಾಶಿಕ ನಿಯಮ) ಬಳಸಿ ದೈನಂದಿನ ಜೀವನದ ಸಮಸ್ಯೆಗಳನ್ನು ಹೇಗೆ ಪರಿಹರಿಸುವುದು ಎಂಬುದನ್ನು ಈ ಅಧ್ಯಾಯವು ವಿವರಿಸುತ್ತದೆ. ಚಿತ್ರಗಳ ಗಾತ್ರದ ಹೋಲಿಕೆ, ಮಿಶ್ರಣಗಳ ತಯಾರಿಕೆ ಮತ್ತು ಸಮಯ-ದೂರದ ಲೆಕ್ಕಾಚಾರಗಳಿಗೆ ಸಮಾನುಪಾತದ ತರ್ಕವನ್ನು ಹೇಗೆ ಅನ್ವಯಿಸಬೇಕು ಎಂಬುದನ್ನು ಇಲ್ಲಿ ಕಲಿಯಬಹುದು.

ಪರೀಕ್ಷೆಗಾಗಿ ನೆನಪಿಡಬೇಕಾದ ಪ್ರಮುಖ ಸೂತ್ರಗಳು

  • ಅನುಪಾತ: ಎರಡು ಪ್ರಮಾಣಗಳ ನಡುವಿನ ಸಂಬಂಧವನ್ನು a:b ಎಂದು ಸೂಚಿಸಲಾಗುತ್ತದೆ.
  • ಸಮಾನುಪಾತ: a:b :: c:d ಆಗಿದ್ದರೆ, a/b = c/d.
  • ಓರೆ ಗುಣಾಕಾರ (Cross Multiplication): a:b :: c:d ಆಗಿದ್ದರೆ, a × d = b × c.
  • ತ್ರೈರಾಶಿಕ ನಿಯಮ (Unitary Method): ಇಚ್ಛಿತ ಫಲಿತಾಂಶ = (ಫಲಿತಾಂಶ × ಇಚ್ಛೆ) / ಪ್ರಮಾಣ.
  • ತಾಪಮಾನ ಪರಿವರ್ತನೆ:
    • °F = (9/5 × °C) + 32
    • °C = 5/9 × (°F - 32)

ಅಭ್ಯಾಸ ಪ್ರಶ್ನೆಗಳು ಮತ್ತು ಪರಿಹಾರಗಳು

ಪ್ರಶ್ನೆ 1: ಸಮಾನುಪಾತದ ಹೇಳಿಕೆಗಳನ್ನು ಗುರುತಿಸಿ

ಪರಿಹಾರ:

  • (i) 4:7 :: 12:21 (4/7 = 12/21 = 4/7) - ಸರಿ
  • (ii) 8:3 :: 24:6 (8/3 ≠ 24/6) - ತಪ್ಪು
  • (iii) 7:12 :: 12:7 - ತಪ್ಪು
  • (iv) 21:6 :: 35:10 (21/6 = 7/2, 35/10 = 7/2) - ಸರಿ
  • (v) 12:18 :: 28:12 - ತಪ್ಪು
  • (vi) 24:8 :: 9:3 (24/8 = 3/1, 9/3 = 3/1) - ಸರಿ

ಪ್ರಶ್ನೆ 2: 4:9 ಕ್ಕೆ ಸಮಾನವಾದ 3 ಅನುಪಾತಗಳು

ಪರಿಹಾರ: ಅನುಪಾತದ ಎರಡೂ ಪದಗಳನ್ನು ಒಂದೇ ಸಂಖ್ಯೆಯಿಂದ ಗುಣಿಸಿ:

  • 4×2 : 9×2 = 8:18
  • 4×3 : 9×3 = 12:27
  • 4×4 : 9×4 = 16:36

ಪ್ರಶ್ನೆ 3: 18:24 ಕ್ಕೆ ಸಮಾನವಾದ ಅನುಪಾತಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ

18:24 ರ ಸರಳ ರೂಪ = 3:4. ಆದ್ದರಿಂದ:

  • 3 : 4
  • 12 : 16 (3×4 : 4×4)
  • 20 : 26.66 (3×6.66 : 4×6.66)
  • 27 : 36 (3×9 : 4×9)

ಪ್ರಶ್ನೆ 8 (ಪುಟ 175): ಭೂಮಿಯ ದೂರದ ಲೆಕ್ಕಾಚಾರ

ಪರಿಹಾರ: 1 ವರ್ಷ = 52 ವಾರಗಳು. 52 ವಾರಗಳಲ್ಲಿ 940 ಮಿಲಿಯನ್ ಕಿ.ಮೀ.

1 ವಾರದಲ್ಲಿ = 940 / 52 = 18.07 ಮಿಲಿಯನ್ ಕಿ.ಮೀ.

ಪ್ರಶ್ನೆ 1 (ಪುಟ 179): ಇಟ್ಟಿಗೆಗಳ ಲೆಕ್ಕಾಚಾರ

ಪರಿಹಾರ: 10 ಅಡಿ ಗೋಡೆಗೆ 1450 ಇಟ್ಟಿಗೆಗಳು. ಒಟ್ಟು ಗೋಡೆಯ ಉದ್ದ = 12+15+9+9+6 = 51 ಅಡಿ.

1 ಅಡಿಗೆ = 1450 / 10 = 145 ಇಟ್ಟಿಗೆಗಳು.

51 ಅಡಿಗೆ = 51 × 145 = 7395 ಇಟ್ಟಿಗೆಗಳು.

ಪ್ರಶ್ನೆ 1 (ಪುಟ 184): 4500 ಅನ್ನು 2:3 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸಿ

ಪರಿಹಾರ: ಒಟ್ಟು ಭಾಗಗಳು = 2+3 = 5.

ಒಂದು ಭಾಗದ ಮೌಲ್ಯ = 4500 / 5 = 900.

ಮೊದಲ ಭಾಗ = 2 × 900 = 1800.

ಎರಡನೇ ಭಾಗ = 3 × 900 = 2700.

ಪ್ರಶ್ನೆ 2 (ಪುಟ 184): ಆಮ್ಲ ಮತ್ತು ನೀರಿನ ದ್ರಾವಣ

ಪರಿಹಾರ: ಅನುಪಾತ 1:5, ಒಟ್ಟು ಭಾಗಗಳು = 6.

ಒಂದು ಭಾಗ = 204 / 6 = 34 ml.

ಆಮ್ಲ = 1 × 34 = 34 ml.

ನೀರು = 5 × 34 = 170 ml.

ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಪ್ರಮುಖ ಸಲಹೆಗಳು

  • ಅಲರ್ಟ್: ಅನುಪಾತಗಳನ್ನು ಹೋಲಿಸುವಾಗ ಯಾವಾಗಲೂ ಅವುಗಳನ್ನು ಮೊದಲು ಸರಳ ರೂಪಕ್ಕೆ (Lowest form) ತನ್ನಿ.
  • ತಪ್ಪು ಮಾಡಬೇಡಿ: ಸಮಯ ಮತ್ತು ವೇಗದ ಲೆಕ್ಕಾಚಾರದಲ್ಲಿ, ವೇಗ ಹೆಚ್ಚಾದಂತೆ ಸಮಯ ಕಡಿಮೆಯಾಗುತ್ತದೆ (ವಿಲೋಮಾನುಪಾತ). ಇಲ್ಲಿ ನೇರ ಸಮಾನುಪಾತದ ಸೂತ್ರವನ್ನು ಬಳಸಬೇಡಿ.
  • ಏಕಮಾನಗಳು: ಲೆಕ್ಕ ಮಾಡುವಾಗ ಎಲ್ಲಾ ಅಳತೆಗಳು ಒಂದೇ ಏಕಮಾನದಲ್ಲಿ (ಉದಾಹರಣೆಗೆ: ಎಲ್ಲವೂ ಮಿಲಿಲೀಟರ್ ಅಥವಾ ಎಲ್ಲವೂ ಲೀಟರ್) ಇವೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
